Af Johan, Lukas og Victor.

Vi har vedlagt et SQL script kaldt "db_cafe".
Opret en MySQL database, som hedder "Cafe" og kør scriptet.
I WebStorm, opret en miljøvariabel til Server\server.js, som hedder
"dbpassword".
Tilføj dit database password til miljøvariablen.
Hvis dit MySQL username ikke er "root" skal du ændre i koden på server.js
på linje 14.

Kør serveren, fyr index.html af og kig rundt!
